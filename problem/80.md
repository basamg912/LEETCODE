```cpp
class Solution {
public:
    int removeDuplicates(vector<int>& nums) {
        int size = nums.size();
        unordered_map<int, int> check;
        vector<int> temp(size);

        int cnt=0;
        for (int i=0; i<size; i++){
            int num = nums[i];
            if (check[num] == 2){
                continue;
            }
            check[num] += 1;
            temp[cnt++] = num;
        }
        nums = temp;
        return cnt;
    }
};
```
Approach
- 처음에는 이전의 중복제거 문제와 동일하게 접근했다. 새롭게 알게된건 unordered_map 은 없는 키에 접근하면 `[]` 알아서 0 으로 값을 채워주는 로직이 있다. value initialization `T{}` 의 경우 0 , 0.0, false 등으로 값을 채워줌
ex) `int n{}` -> 0 으로 기본값 , `vector<int> v(5)` -> 5개 원소가 0 으로 초기화된 상태, `int *p{}` -> nullptr 초기화

위 접근법은 runtime, memory 측면에서 많이 안 좋은 solution 이였다.
