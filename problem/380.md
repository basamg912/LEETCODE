Approach
- Remove 를 O(1) 에 시행하고, 모든 원소에 대해서 동일 확률로 sampling -> return 하는 getRandom 함수 구현. 후자는 rand() 를 몰라서 생각해내는게 어려웠다.
- 처음에는 unordered_map 를 is_exist 로 nums 저장하는 vector 에 값이 있는지 확인 용도로 사용했었는데, 값의 index 를 저장하도록 <int, int> 로 map 을 구현하는게 핵심이였다.
- 추가로 c++ stl set 은 트리구조로 값을 저장해서 값 탐색, 삭제가 O(nlogn) (red-black tree 구조)
