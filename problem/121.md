Approach
```cpp
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int size = prices.size();
        int max = 0;
        for (int i=0; i<size; i++){
            for (int j=i+1; j<size; j++){
                int v = prices[i] - prices[j];
                max = (max > v) ? max : v;
            }
        }
        return max;
    }
};
```
- time limit 초과 발생
- input 원소가 매우 많을때 발생 O(n^2)

O(n) Approach
```cpp
class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int MIN_PRICE = INT_MAX;
        int MAX_PROFIT = 0;

        for (price : prices){
            MIN_PRICE = min(MIN_PRICE, price);
            MAX_PROFIT = max(MAX_PROFIT, price - MIN_PRICE);
        }
        return MAX_PROFIT
    }
};
```
- 앞에서부터 훑으면서 MIN_PRICE 업데이트, MIN_PRICE 로 PROFIT 계산
- 모든 경우에 대해서 profit 계산 없이 한번만 훑어도 계산 가능 (iterative 하게 max profit 계산하는 방식)
- `[8 6 4 9 8 1 3 4]`
  - 최대 이익 : 4에 사서 9에 팔기 (min 4 -> max 9)
  - global min 은 1 -> 하지만 (1, 3 -> profit 2 / 1, 4 -> profit 4) 라서 max profit 업데이트 유발은 안함
