`14. https://leetcode.com/problems/longest-common-prefix/`

문제 접근 과정
- DP 방법 비스무리하게 접근하면 되겠다 싶었다. 문제 조건이 최대 200개의 string 을 담고 있는 vector 를 처리해야하는데 time complexity 를 생각했을때, 모든 쌍 비교는 우선 별로라고 생각했고 첫번재 문자를 기준으로 각 string 에 대해 순차적으로 비교해가면서 `vector<bool> dp(strs[0].size, false)` 를 true 로 채워나가면서 해볼까? 싶은 생각을 했다.
- 굳이 vector 사용하지않고 바로 2중 for 문으로 풀이했고, flag 를 설정해서 length 를 증가시키고 마지막에 substr 함수로 부분문자열을 반환하도록 설계했다.
- 헤맨 부분은 `else flag` 부분? 비교하다가 중간에 일치하지않는 경우 for 문을 탈출해야하는데 어디서 break 를 선언해야하지? 를 조금 헤맸다.
