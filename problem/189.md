Approach
- 원소를 오른쪽으로 밀어주면 되는거니깐 index 를 바꿔주는 방식을 적용해봄
- 헷갈렸던 부분은 index target 을 어떻게 설정하는지. 처음에는 `temp[i] = nums[idx]` 로 했었는데, 직관적으로 생각하면 잘못된 방식이였음

```cpp
class Solution {
public:
    void rotate(vector<int>& nums, int k) {
        int size = nums.size();
        vector<int> temp(size);
        for (int i=0; i<size; i++){
            int idx = (i+k) % size;
            temp[idx] = nums[i];
        }
        nums = temp;
    }
};
```
