Problem Description
- 중복되는 글자 없이 sub string 의 maxlen 을 찾는 문제

Approach (step to step)
- left 와 right 로 window 설정 후 문자열 끝까지 O(n) 에 해결
  - or) DP 로 O(n^2) 도 가능할듯
- 알파벳 갯수 (26) 만큼의 `vector<bool>` 선언 후 문자를 본 적 있는지? 를 확인 용도로 사용 → 알파벳 본 적 있는가 + 어느 위치에서 봤는가? 를 확인해야함 → `unordered_map` 사용으로 대체
- `unordered_map<char, int>` 해시 테이블로 구현됨 → key-value 넣으면 같은 key 원소끼리 연결시켜서 테이블구조 생성. `count()` 는 특정 키가 해시테이블에 존재하는지를 확인. c++20 부터는 `contains()` 로 의도를 명확히함

Declaration + Initialize
1. `string s("HelloWorld")` : Direct Initialize
2. **`string s{"HelloWorld"}` : Uniform Initialize**
3. `string s = "HelloWorld"` : Copy Initialize

Memory 구조 이해하기 (Copy Initialize 를 이용한)
- Data 영역 : `"HelloWorld"` 라는 문자열은 컴파일 타임에 데이터영역에 공간할당
- `string s` 라는 변수는 컴파일 타임에 스택 영역에 공간할당
- 데이터 영역에 있는 "HelloWorld" 를 힙영역으로 복사, 이후 `string s` 의 스택 영역 안에 있는 `char* ptr` 가 힙영역의 문자열 주소를 가리킴
